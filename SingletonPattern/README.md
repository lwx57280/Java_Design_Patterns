# Java_Design_Patterns
单例设计模式
------------
    什么是单利模式？
    	单利模式是一种对象创建型模式，使用单利模式可以确保一个类只生成唯一实例。也就是说在整个程序空间中，该类只存在一个实例对象。
    	1、私有的构造。
    	2、私有的静态属性类型自己。
    	3、公有的静态方法返回值类型是类自身的实例。
    单利模式的好处节省内存资源，它限制了实例的个数，有利于java垃圾回收。
    
    注意：
    
    1、单例类只能有一个实例。
    2、单例类必须自己创建自己的唯一实例。
    3、单例类必须给所有其他对象提供这一实例。
